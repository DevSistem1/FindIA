# FinAdmin IA Pro

Sistema simples para pequenos negócios com:

- Financeiro
- Estoque
- Vendas
- Clientes
- Agendamentos
- Relatórios
- Assistente IA simulada com respostas baseadas nos dados

## Como rodar

```bash
npm install
npm run dev
```

Abra no navegador:

```txt
http://localhost:3000
```

Os dados ficam salvos no `localStorage` do navegador. Não precisa configurar banco de dados nesta versão.
